# Aditya Maurya — Developer Portfolio

A modern developer portfolio showcasing my projects, technical interests, open-source work, and journey in software development.

The portfolio is built around a clean, content-focused interface, with a future Ubuntu-inspired interactive developer environment for exploring the same portfolio content.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Canvas API
- Lucide React

## Featured Projects

### RepoJet

A developer CLI tool written in Go that analyzes software repositories, detects package managers, and helps prepare projects for execution.

**Tech:** Go, Cobra, Docker, Git

### BinMap Tracker

A smart waste-management application for mapping, discovering, and reporting public dustbins.

**Tech:** Next.js, Supabase, Leaflet

### Technical Services Management System

A services marketplace supporting clients, freelancers, projects, proposals, reviews, and payments.

**Tech:** React, JavaScript, SQL

## Features

- Responsive single-page portfolio
- Data-driven project architecture
- Active-section navigation
- Light and dark themes
- Canvas-based animated background
- Accessible and responsive UI
- Future Ubuntu-inspired portfolio experience

## Getting Started

```bash
git clone <repository-url>
cd FinalPortfolio

pnpm install
pnpm dev
```
